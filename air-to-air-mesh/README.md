# Air-to-Air Mesh Network (AAMN)

This module implements the AirMesh protocol for direct RF-based communication between aircraft, enabling instant position sharing, autonomous coordination, and distributed autopilot systems.

## Features

- Instant transmission of position, velocity vector, and maneuvers.
- Autonomous air traffic coordination.
- Distributed collective autopilot system.
- Zero dependency on ground services.

## Structure

- `src/`: C/C++ source code for the protocol stack.
- `docs/`: Documentation and specifications.

## Building

Use CMake to build the module. Refer to the top-level README for general instructions.
