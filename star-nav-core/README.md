# StarNav Core

This module implements celestial navigation using star cameras and constellation recognition for precise positioning, compensating for aircraft motion.

## Features

- Star camera integration.
- Constellation recognition and catalog correlation.
- Motion compensation.
- Precise coordinate output.

## Structure

- `src/`: C++ code for navigation algorithms.
- `docs/`: Documentation and astronomical details.

## Building

Use CMake to build the module. Refer to the top-level README for general instructions.
